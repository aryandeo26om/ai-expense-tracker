# 💡 FinTrack – AI Expense Tracker

An **AI-powered expense tracking application** built using **Streamlit, Python, and Machine Learning**.
It helps users track daily expenses, analyze spending patterns, and get intelligent insights.

---

## 📌 Features

### 🔐 Authentication

* User Registration and Login system
* Secure password hashing using Werkzeug
* Session-based user access

---

### 📊 Dashboard

* Total spending overview
* Number of entries
* Category tracking
* Average expense calculation
* Smart spending alerts

---

### ➕ Expense Management

* Add new expenses
* Categorize expenses
* Delete expenses
* Real-time updates

---

### 📈 Analytics

* Category-wise expense visualization
* Interactive charts
* AI-based future expense prediction

---

### 🤖 AI Assistant

Ask simple queries like:

* "highest expense"
* "total spending"
* "top category"

Get instant insights from your data.

---

### 📥 Export Feature

* Download your expense data as CSV
* Useful for reporting and backups

---

## 🧠 AI Model

The application uses a simple **Linear Regression model** to predict future expenses based on historical data.

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Database:** SQLite
* **Machine Learning:** Scikit-learn
* **Authentication:** Werkzeug

```bash
git clone https://github.com/aryandeo26om/ai-expense-tracker.git
cd ai-expense-tracker

pip install -r requirements.txt
streamlit run streamlit_app.py
```
